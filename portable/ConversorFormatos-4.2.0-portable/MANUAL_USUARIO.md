# Manual de Usuario

Documento vivo del proyecto. Este manual se mantiene en español y describe el uso general de la aplicación.

## Introducción

Conversor de Formatos Tabulares es una aplicación de escritorio para convertir archivos tabulares entre distintos formatos comunes.

## Versión actual

5.0.0

## Fecha de última actualización

2026-04-15

## Idiomas de la interfaz

La aplicación ahora permite usar la interfaz en:

- Español
- Inglés

El idioma puede cambiarse desde la ventana principal y, cuando es posible, la preferencia se conserva entre sesiones.

Nota:

- En esta fase la interfaz es bilingüe.
- La documentación del proyecto y este manual siguen en español.

## Uso básico

1. Selecciona o arrastra un archivo compatible.
2. Revisa la vista previa.
3. Elige el formato de salida.
4. Presiona `Convertir`.
5. Guarda el resultado.

## Modalidades de distribución

### Versión portable

- No requiere instalación.
- Puede ejecutarse desde una carpeta o memoria USB.

### Versión instalable

- Requiere ejecutar un instalador de Windows.
- Crea accesos directos y una experiencia más tradicional.
